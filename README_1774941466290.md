# 🫀 Edge AI-Based Real-Time ECG Analysis System

## Overview
An interdisciplinary project combining **AI + Healthcare + Embedded Systems** for real-time ECG signal analysis. This system demonstrates stress detection, emotion classification, and arrhythmia detection using edge AI techniques - completely simulated in software without requiring any hardware.

## 🎯 Key Features

### 1. **Real-Time ECG Signal Simulation**
- Generates realistic ECG waveforms using NeuroKit2
- Multiple physiological states: Normal, Atrial Fibrillation, Bradycardia, Tachycardia, Stressed, Relaxed
- Configurable noise levels and signal duration
- Automatic R-peak detection and heart rate calculation

### 2. **Arrhythmia Detection (ML-Based)**
- Pre-trained Random Forest and SVM classifiers
- Detects 5 cardiac conditions:
  - Normal Sinus Rhythm
  - Atrial Fibrillation
  - Bradycardia
  - Tachycardia
  - Ventricular Tachycardia
- Confidence scores and probability distributions
- Feature extraction: RR intervals, QRS duration, heart rate variability

### 3. **HRV-Based Stress Analysis**
- **Time Domain Metrics**: SDNN, RMSSD, pNN50
- **Frequency Domain Analysis**: LF power, HF power, LF/HF ratio
- Stress level classification (Low/Moderate/High)
- Autonomic Nervous System (ANS) balance assessment
- Recovery status evaluation

### 4. **Emotion Classification**
- Infers emotional state from cardiac patterns
- 5 emotion states: Calm, Anxious, Stressed, Relaxed, Excited
- Valence-Arousal model implementation
- Rule-based classification using HRV and heart rate

### 5. **Edge AI Performance Metrics**
- Simulates deployment on edge devices:
  - ARM Cortex-M4
  - ESP32
  - Raspberry Pi
  - Edge TPU
- Real-time performance metrics:
  - Inference latency
  - Model size and memory usage
  - CPU/Memory utilization
  - Power consumption
  - Battery life estimation
- Model optimization techniques (quantization, compression)

### 6. **Interactive Dashboard**
- Multi-tab interface for comprehensive analysis
- Real-time ECG waveform visualization
- HRV Poincaré plots and distributions
- Historical data tracking
- Export functionality (CSV/JSON)
